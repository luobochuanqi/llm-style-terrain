# LandCraft论文地形生成全流程（纯地形部分，不含地物）
所有信息均严格提取自论文原文，仅保留与**裸地地形高程生成、地形网格构建**相关的内容，排除城镇、植被、水体等地物生成环节。

## 一、训练阶段：地形相关数据集的程序化合成
论文通过自研程序化算法自动合成训练MapGen所需的地形相关数据，无外部地形数据集依赖：
1.  先通过**随机游走+元胞自动机算法**合成语义布局图\(M_l\)，标记出山地区域的空间范围。
2.  针对\(M_l\)中标记的**山地区域**，采用**梯度场基方法**模拟地形高程特征；针对山地区域外的其他区域，采用**Perlin噪声**生成基础地形高程，最终得到配对的高精度高度场图\(M_h\)。
3.  同步提取山地元素的三元组\(E_i=(c_i, P_i, Attr_i)\)（类别、空间坐标、局部属性）与全局地形参数。
4.  基于上述信息通过GPT-4o生成对应文本描述，人工校对后形成完整样本，最终合成2万条训练样本+2千条测试样本用于MapGen训练。

## 二、推理阶段：文本到裸地地形的三级生成流程
### （一）第一阶段：LLM驱动的地形初始具象化
1.  **输入**：用户文本提示。
2.  **方法**：基于GPT-4o通过上下文学习（ICL），从文本中提取与地形直接相关的两类信息：
    - 地形元素三元组：针对山地类地形元素，生成标准化三元组\(E_i=(c_i, P_i, Attr_i)\)，其中\(c_i\)为"mount"（山地）类别，\(P_i\)为山地所在区域的2D bounding box（左上角与右下角顶点的(x,y)坐标），\(Attr_i\)为山地的局部外观属性；
    - 全局地形参数\(F\)：描述整个地形的基础特征，如基础地表色调。
3.  **输出**：山地元素的空间位置与属性、全局地形参数，形成地形的粗粒度规划。

### （二）第二阶段：细粒度地形高度图生成（核心环节）
基于LLM输出的粗粒度规划，通过两个串联的MapGen模块依次生成语义布局图与高度图，为最终地形构建提供精准的高程基底。
1.  **前置步骤**：将LLM提取的所有地形元素的数值空间位置绘制到2D图像\(M_e\)上，作为MapGen的初始输入。
2.  **MapGen基础架构**：采用Diffusion Transformer（DiT）作为基础架构，通过CLIP文本编码器提取文本嵌入，经多头交叉注意力注入DiT模块；采用**布朗桥扩散模型（BBDM）** 替代传统潜在扩散模型，其扩散过程定义为：
    \[x_t = (1-\frac{t}{T})M + \frac{t}{T}M' + \gamma\]
    其中\(\gamma \sim N(0, \sigma_tI)\)为噪声项，基础扩散损失为：
    \[L_{bbdm} = \|\epsilon_\theta(x_t, t, \tau) - \epsilon_t\|^2, \epsilon_t = \frac{t}{T}(M' - M) + \gamma\]
3.  **第一步：生成地形语义布局图\(M_l\)**
    - **输入**：文本提示\(\tau\) + 初始位置图像\(M_e\)；
    - **损失函数**：总损失\(L_{M_l}=L_{bbdm}+\lambda_1L_{lc}\)，其中\(L_{lc}\)为布局一致性损失，通过计算所有语义类别的Soft-IoU得分实现：
      \[L_{lc}=1-\frac{1}{|C|}\sum_c\frac{(\hat{B}_c \cdot B_c)}{(\hat{B}_c + B_c - \hat{B}_c \cdot B_c)}\]
      式中\(\hat{B}_c\)为预测布局图中类别c的二值掩码，\(B_c\)为LLM输出的山地元素位置对应的二值掩码（bounding box内为1，其余为0）；
    - **输出**：包含山地语义区域的细粒度地形布局图\(M_l\)。
4.  **第二步：生成地形高度场图\(M_h\)**
    - **输入**：文本提示\(\tau\) + 地形语义布局图\(M_l\)；
    - **损失函数**：总损失\(L_{M_h}=L_{bbdm}+\lambda_2L_{grad}\)，其中\(L_{grad}\)为梯度损失，用于约束地形高程的几何连续性：
      \[L_{grad}=\|\nabla_x\hat{M}_h - \nabla_xM_h\|_1 + \|\nabla_y\hat{M}_h - \nabla_yM_h\|_1\]
      式中\(\nabla_x、\nabla_y\)分别为水平、垂直梯度算子，\(\hat{M}_h\)为预测高度图，\(M_h\)为真实高度图；
    - **输出**：与语义布局对齐的高精度地形高度场图\(M_h\)。

### （三）第三阶段：裸地地形3D网格构建
1.  **输入**：细粒度地形高度场图\(M_h\)。
2.  **方法**：基于高度场图，程序化构建拓扑正确的**3D三角网格裸地地形表面**，无任何外部地形资产依赖。
3.  **输出**：可直接用于后续地物叠加的工业级裸地地形网格资产，支持无缝导出至外部游戏引擎与建模软件。
